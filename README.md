# 🍽️ Recipe App

A modern and responsive Recipe App built using React that allows users to search for delicious recipes, view detailed cooking instructions, and explore different cuisines using a public Recipe API.

## 🚀 Features

- 🔍 Search recipes by name
- 🍲 View recipe details
- 📝 Ingredients list
- 👨‍🍳 Step-by-step cooking instructions
- 🖼️ Recipe images
- 📱 Responsive design
- ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- REST API
- Git & GitHub

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/recipe-app-1.git
```

### Navigate to the project

```bash
cd recipe-app-1
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm start
```

The application will start on:

```
http://localhost:3000
```
